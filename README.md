# HTML5 Sign-Up Form
### Built With
- HTML5
- CSS3 (Grid & Flexbox)

This is a sign-up form I built to practice working with HTML5 forms. It includes multiple input validations and pseudo-class styling. The layout is created using a combination of CSS Grid and Flexbox.
